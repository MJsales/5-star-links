How to Use

1

Download the Tool

Click the download button for your platform. Standalone app — no Python or Node.js needed.

2

Install FFmpeg & yt-dlp (if needed)

Windows: Run winget install ffmpeg yt-dlp in PowerShell
Mac: Run brew install ffmpeg yt-dlp
Linux: Run sudo apt install ffmpeg && pip install yt-dlp

3

Run It

Windows: Unzip 5starclips.zip and double-click 5starclips.exe — opens a terminal with a purple banner. Paste a YouTube URL and it auto-clips.
Mac: Unzip 5starclips-mac.zip. First run: right-click 5star-splicer-mac → Open (to bypass Gatekeeper), then click Open again. After that, just double-click it — it opens the app in your browser automatically.

4

Paste a YouTube URL

The tool downloads the video, transcribes it with AI (if Whisper is installed), finds viral moments, and clips them automatically.

5

Get Your Clips

Find your clips in the 5star_clips folder. Ready to post on TikTok, Reels, Shorts!

System Requirements

  • Windows 10+ or macOS 11+ (Intel or Apple Silicon via Rosetta)
  • FFmpeg & yt-dlp (auto-detected, or auto-installed via WinGet on Windows / Homebrew on Mac)
  • Optional: Python + openai-whisper for AI-powered viral detection
  • ~200MB disk space for video downloads
  • Works 100% locally. Nothing sent to cloud.
